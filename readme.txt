== Description ==

Permet de récupérer les informations de version de WordPress ainsi que des plugins installés.

La récupération au format XML se fait à partir d'une URL, que l'on peut définir.
 
Il est également possible d'indiquer un mot de passe.

== Installation ==

Upload the Prelude Version plugin to your blog, Activate it.

See the Settings page for more informations.
